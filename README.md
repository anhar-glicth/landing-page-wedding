# 🌸 Landing Page Jasa Undangan Pernikahan

Landing page ini dirancang khusus untuk menawarkan jasa desain undangan pernikahan digital dengan estetika premium, elegan, dan profesional.

## 🏗️ Struktur Proyek
- `index.html`: Struktur utama halaman (Semantic HTML5).
- `style.css`: Desain visual dengan tema **Cream & Gold**.
- `script.js`: Interaksi dinamis dan animasi.
- `hero.png`: Gambar latar belakang hero yang dihasilkan secara AI.

## ✨ Fitur Utama
1. **Hero Section Profesional**: Menampilkan value proposition layanan Anda secara jelas.
2. **Katalog Portofolio**: Menampilkan 3 link demo yang Anda berikan dalam bentuk kartu yang elegan.
3. **Daftar Harga Transparan**: Paket **Statis (90k)** dan **Dinamis (190k)** dengan daftar fitur yang jelas.
4. **Integrasi WhatsApp**: Tombol CTA (Call to Action) yang langsung terhubung ke WhatsApp.
5. **Animasi Smooth**: Menggunakan library **AOS** untuk efek *scroll-triggered animations*.
6. **Responsive Design**: Tampilan yang sempurna di perangkat mobile maupun desktop.

## 🎨 Kustomisasi
- **Warna**: Ubah variabel `:root` di `style.css` untuk mengganti tema warna.
- **Gambar Portofolio**: Ganti URL `background-image` di bagian `portfolio-card` dengan screenshot asli dari undangan Anda.
- **Link WhatsApp**: Ubah nomor telepon pada bagian `btn-whatsapp` di `index.html`.
